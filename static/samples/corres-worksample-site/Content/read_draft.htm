Read Correspondence and Draft Response

Previous Procedure

Deliver Correspondence to Addressees

Responsible Person

Addressee

Tools Needed

  • Computer with Microsoft Word® software and connection to office's network folder
  • Printer
  • Letter template office-init-resp-2011.dot

Procedure

Step Action
1. Read the correspondence, making notes of any questions and requests.
2. Open the template office-init-resp-2011.dot.
3. Follow the template instructions to write a response letter. Make sure to address any questions and requests in the original correspondence.
4. Proofread the response letter draft for typos and misspellings.
5. Save the draft in the network folder. The file name format is YYYY-MM-DD_IDnumber_init.doc.
6. Print the response draft and bring it (with the original correspondence) to the Office Administrative Assistant.
