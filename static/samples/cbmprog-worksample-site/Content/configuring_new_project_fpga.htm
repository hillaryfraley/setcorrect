1.2 Configuring a New Project

  1. Turn on the FlashPro programmer.
  2. On your computer, double-click the flashpro.exe application. The FlashPro window opens.
  3. In the FlashPro window, click the New Project button. The New Project window opens (Figure 1).

Figure 1 New Project Window

  1. Type a name for the new project in the Project Name field.
  2. Select the Browse... button and navigate to the location where you want to store the new project file.
  3. Select the Single Device option.
  4. Click the OK button.
  5. In the FlashPro window, select Tools > Programmer Settings. The Programmer Settings window opens.
  1. Check the boxes to select the following options (Figure 2):
    • Set Vpp
    • Set Vpn
    • Set Vdd
    • Set Vddp
    • Drive TRST

Figure 2 Selecting Programmer Settings

  1. Click the OK button. The FlashPro [CBM_FIRMWARE_FPGA] window opens (Figure 3).

Figure 3 FlashPro [CBM_FIRMWARE_FPGA] Window

  1. Click the Configure Device button. The Single Device Configuration panel opens.
  2. Click the View Programmers button. The Programmer List panel opens.
  3. In the Programmer List panel, click the Refresh/Rescan for Programmers button (Figure 4).

Figure 4 Refresh/Rescan for Programmers

  1. Close the FlashPro [CBM_FIRMWARE_FPGA] window.
  2. Close the FlashPro window.

This completes the procedure for configuring a new project. The next procedure is 1.3 Programming the FPGA.
