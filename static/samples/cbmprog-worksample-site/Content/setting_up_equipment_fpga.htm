1.1 Setting up Equipment for FPGA Programming

  1. Gather the following items:
    • Controller area network (CAN) bus master (CBM) printed circuit board (PCB) assembly (part number xxxxx)
    • Microsemi FlashPro programmer (part number xxxxx)
    • CBM standard test and programming language (STAPL) file (filemame.stp)
    • Your computer
  2. Connect the FlashPro programmer harness to the 26-pin Joint Test Action Group (JTAG) header on the CBM PCB.
  3. Connect the USB cable from the FlashPro programmer to your computer.
  4. Follow the Microsemi instructions to install the FlashPro programmer.
